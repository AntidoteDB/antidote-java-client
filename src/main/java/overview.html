This is a library for working with the Antidote Database.

Installation

The library is available on Maven Central, so you can simply add it as a dependency to your project (make sure to update the version used in this example):

Maven

<dependency>
	<groupId>eu.antidotedb</groupId>
	<artifactId>antidote-java-client</artifactId>
	<version>0.0.5</version>
</dependency>

Gradle

compile group: 'eu.antidotedb', name: 'antidote-java-client', version: '0.0.5'

Local installation

You can also install the latest version of the library from the source code instead of relying on public Maven repositories: 
git clone https://github.com/SyncFree/antidote-java-client.git
cd antidote-java-client
./gradlew install

Usage

Documentation about using the Client can be found in the documentation of the eu.antidotedb.client package.
